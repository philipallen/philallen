JavaScript in Internet Explorer

The Trident engine is used by Microsoft in their Internet Explorer browsers. It has become out of date with web standards, so the below table highlights some major differences between Trident and today's more commonly used rendering engine, WebKit.

Microsoft's latest browser, Edge, uses a fork of Trident, called EdgeHTML, which follows web standards more closely.
